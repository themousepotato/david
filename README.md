# david

> Now I've heard there was a secret chord  
> That David played and it pleased the Lord

This project uses:

- [pnpm](https://pnpm.io/) for package management
- [vite](https://vitejs.dev/) for build tooling

## Dev

- Install dependencies
  `pnpm i`

- Run dev server
  `pnpm dev`

- Build project
  `pnpm build`
